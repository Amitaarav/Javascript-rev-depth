## Callbacks

  Kisi cheez ko vaapas se call karna: I will call u later

  Def: A function that is passed as an argument to another function and is executed later, often after a specific task is completed

  // Process of making tea

  Step-3: Once water start boilling add tea leaves
  Step-4: Add Sugar
  Step-5: After some time add Ginger or whatever u want to add
  Step-6: Boil it for few mins and then pour it
